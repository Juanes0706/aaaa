Información del Proyecto

Descripción General

Este proyecto es una API RESTful completa para la gestión de inventarios y ventas, desarrollada con FastAPI y PostgreSQL. El sistema permite manejar productos, clientes, ventas, categorías y usuarios con funcionalidades avanzadas como subida de imágenes y auditoría de eliminaciones.

Arquitectura del Sistema

  • Backend: FastAPI con SQLAlchemy ORM
  • Base de Datos: PostgreSQL
  • Autenticación: JWT tokens
  • Almacenamiento de Imágenes: Supabase
  • Documentación: Swagger UI y ReDoc

Modelos de Datos

Usuario

  • ID (UUID)
  • Nombre (string)
  • Cédula (string)
  • Correo (string, único)
  • Rol (enum: admin, vendedor, cliente)
  • Fecha de creación (datetime)

Producto

  • ID (UUID)
  • Nombre (string)
  • Descripción (text)
  • Cantidad (integer)
  • Valor unitario (decimal)
  • Valor mayorista (decimal)
  • Categoría ID (UUID, FK)
  • Imagen URL (string)
  • Fecha de creación (datetime)

Cliente

  • ID (UUID)
  • Nombre (string)
  • Cédula (string)
  • Correo (string)
  • Teléfono (string)
  • Dirección (text)
  • Fecha de creación (datetime)

Venta

  • ID (UUID)
  • Cliente ID (UUID, FK)
  • Producto ID (UUID, FK)
  • Cantidad (integer)
  • Precio unitario aplicado (decimal)
  • Total (decimal, calculado)
  • Fecha de creación (datetime)

Categoría

  • ID (UUID)
  • Nombre (string)
  • Código (string)
  • Imagen URL (string)
  • Fecha de creación (datetime)

Endpoints Principales

  • GET / - Estado general de la API
  • GET /health - Verificación de salud
  • GET /docs - Documentación Swagger UI
  • GET /redoc - Documentación ReDoc
